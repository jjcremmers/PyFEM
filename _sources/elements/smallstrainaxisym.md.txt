# SmallStrainAxiSym

The `SmallStrainAxiSym` element performs axisymmetric small strain analysis for rotationally symmetric structures.

## Overview
- **Element type:** `SmallStrainAxiSym`
- **Supports:**
  - Axisymmetric kinematics with small strains
  - Linear and mildly nonlinear analysis

## Parameters
### Mandatory
- `type`: Must be set to `"SmallStrainAxiSym"`
- `material`: Material block appropriate for axisymmetric small strain analysis.
