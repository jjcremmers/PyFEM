# FiniteStrainAxiSym

The `FiniteStrainAxiSym` element performs axisymmetric finite strain analysis for rotationally symmetric structures undergoing large deformations.

## Overview
- **Element type:** `FiniteStrainAxiSym`
- **Supports:**
  - Axisymmetric kinematics with finite strains
  - Nonlinear analysis for large displacements

## Parameters
### Mandatory
- `type`: Must be set to `"FiniteStrainAxiSym"`
- `material`: Material block appropriate for axisymmetric finite strain analysis.
