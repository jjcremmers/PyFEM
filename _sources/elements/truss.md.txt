# Truss

The `Truss` element models axial members with only axial stiffness and no bending resistance, suitable for pin-jointed truss structures.

## Overview
- **Element type:** `Truss`
- **Supports:**
  - Axial-only stiffness
  - Pin-jointed structures and Riks analysis examples

## Parameters
### Mandatory
- `type`: Must be set to `"Truss"`
- `material`: Material block with axial properties (e.g., `E`, area via section or mesh).
