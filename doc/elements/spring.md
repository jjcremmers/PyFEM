# Spring

The `Spring` element models axial spring behavior between nodes and can be used to add elastic supports or connections in structural models.

## Overview
- **Element type:** `Spring`
- **Supports:**
  - Axial spring stiffness
  - Used alongside truss elements in Riks-type analyses

## Parameters
### Mandatory
- `type`: Must be set to `"Spring"`
- `material`: Spring stiffness and properties (depending on input format).
