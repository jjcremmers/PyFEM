SandwichCore

The SandwichCore model provides orthotropic-like stiffness tailored for sandwich cores with dominant through-thickness properties and reduced in-plane stiffness.

Overview

Material type: SandwichCore

Parameters

Mandatory Parameters

Parameter

Description

type

Must be set to "SandwichCore"

E3

Through-thickness Young’s modulus

G

Base shear modulus

Optional Parameters

Parameter

Description

G13 / G23

Shear moduli (defaults to G if not set)

factor

Reduction factor for in-plane stiffness (default 0.001)
